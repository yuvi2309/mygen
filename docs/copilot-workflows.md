# Copilot Workflows for This Project

This workspace includes a tailored Copilot asset setup for building AI-native products with strong engineering discipline.

## What is included

### Instructions
- AI product engineering standards
- Documentation-first guidance
- Testing and quality rules
- Secure coding guidance
- Performance guidance
- Next.js and Tailwind guidance

### Prompts
- Product specification creation
- Implementation planning
- Architecture decision record creation
- Feature documentation generation
- Debug issue workflow
- Refactor code workflow

### Agents
- AI Product Architect
- Specification Writer
- Feature Implementer
- Engineering Reviewer
- Debug Specialist

### Chat modes
- AI Product Architect
- Specification Writer
- Engineering Reviewer

### MCP
- SigMap for codebase-aware retrieval
- Omni for noise-reduced terminal workflows

## Suggested workflow
1. Start with the AI Product Architect or Specification Writer agent.
2. Create a spec and an implementation plan before building.
3. Move to the Feature Implementer agent for delivery work.
4. Use the Engineering Reviewer or Debug Specialist agent as needed.
5. Keep docs, ADRs, and verification updated alongside code.
